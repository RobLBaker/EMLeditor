# Returns the publisher information

`get_publisher()` returns a list that includes all the information about
the publisher stored in EML.

## Usage

``` r
get_publisher(eml_object)
```

## Arguments

- eml_object:

  is an R object imported (typically from an EML-formatted .xml file)
  using EmL::read_eml(, from="xml").

## Value

List.

## Examples

``` r
if (FALSE) { # \dontrun{
get_publisher(eml_object)
} # }
```
